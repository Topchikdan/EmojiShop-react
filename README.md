## О проекте

Небольшой интернет-магазин эмодзи-смайликов на React. Товары загружаются с внешнего API, корзина сохраняется между перезагрузками страницы, есть поиск, страница карточки товара и форма обратной связи.

---

## Возможности

- Каталог товаров: карточки смайликов с ценой и описанием, загружаются с [mockapi.io](https://mockapi.io)
- Поиск: фильтрация по названию и описанию в реальном времени
- Карточка товара: отдельная страница по клику на товар (`/product/:id`)
- Корзина: добавление, изменение количества, удаление товаров; сохраняется в `localStorage`
- Тосты: всплывающие уведомления при добавлении товара в корзину
- Контакты: страница с контактной информацией и формой обратной связи
- Адаптивная вёрстка: корректно отображается на десктопе, планшете и мобильных

## Быстрый старт

Нужен [Node.js](https://nodejs.org/) версии 18 или новее.

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Topchikdan/EmojiShop-react.git
cd EmojiShop-react

# 2. Установите зависимости
npm install

# 3. Запустите в режиме разработки
npm run dev
```

Откройте адрес из терминала (обычно http://localhost:5173).


## API

Данные о товарах берутся с публичного mock-API:

```
https://6a3c21e6e4a07f202e167825.mockapi.io/smiles
```

Формат одного товара:

```json
{
  "id": "1",
  "title": ":)",
  "description": "Весёлый смайлик, наделяет улыбкой",
  "price": 10
}
```

---

## Хранение корзины

Содержимое корзины хранится в `localStorage` под ключом `smiles-shop:cart` в виде массива товаров с дополнительным полем `amount` (количество). Корзина сохраняется между сессиями без необходимости в бэкенде.

---

## Важно: иконки

Проект использует иконки Material Symbols Outlined. Чтобы они отображались корректно, в `index.html` должна быть подключена таблица стилей:

```html
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined&display=swap">
```

Без этой строки иконки (корзина, поиск, крестик и т.д.) будут отображаться как обычный текст.
