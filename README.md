# Emodji Shop — магазин смайликов

Небольшой интернет-магазин эмодзи-смайликов на React. Товары загружаются с внешнего API, корзина сохраняется между перезагрузками страницы, есть поиск, страница карточки товара и форма обратной связи.

## Возможности

- 🛍️ **Каталог товаров** — карточки смайликов с ценой и описанием, загружаются с [mockapi.io](https://mockapi.io)
- 🔍 **Поиск** — фильтрация по названию и описанию в реальном времени
- 🧾 **Карточка товара** — отдельная страница по клику на товар (`/product/:id`)
- 🛒 **Корзина** — добавление, изменение количества, удаление товаров; сохраняется в `localStorage`
- 💬 **Тосты** — всплывающие уведомления при добавлении товара в корзину
- 📨 **Контакты** — страница с контактной информацией и формой обратной связи
- 📱 **Адаптивная вёрстка** — корректно отображается на десктопе, планшете и мобильных

## Стек

- [React](https://react.dev/)
- [React Router](https://reactrouter.com/) — маршрутизация
- [Vite](https://vitejs.dev/) — сборка проекта
- SCSS Modules — стилизация компонентов
- [Material Symbols](https://fonts.google.com/icons) — иконки

## Структура проекта

```
src/
├── App.jsx                  # корневой компонент, роутинг
├── main.jsx                 # точка входа, подключение стилей и роутера
├── components/
│   ├── header/               # шапка сайта (навигация + иконка корзины)
│   ├── footer/                # подвал сайта
│   ├── product/                # карточка товара в каталоге
│   ├── productCard/             # страница отдельного товара
│   ├── products/                 # каталог товаров (загрузка, поиск, сетка)
│   └── toast/                     # всплывающее уведомление
├── pages/
│   ├── home/                # главная страница (каталог)
│   ├── cart/                 # страница корзины
│   └── contacts/              # страница контактов
├── hooks/
│   ├── useCart.js            # логика корзины + синхронизация с localStorage
│   └── useToast.js            # логика показа тост-уведомлений
└── styles/
    ├── variables.scss        # цвета, отступы, тени, breakpoints
    ├── fonts.scss              # подключение Google Fonts
    ├── global.scss              # глобальные стили и общие классы (.state, .toast, .spinner)
    └── index.scss                 # точка входа стилей
```

## Установка и запуск

# установить зависимости
npm install

# запустить дев-сервер
npm run dev

# собрать продакшн-версию
npm run build

# просмотреть собранную версию
npm run preview

После `npm run dev` проект будет доступен по адресу, который выведет Vite в терминал (обычно `http://localhost:5173`).

## Важно: иконки

Проект использует иконки **Material Symbols Outlined**. Чтобы они отображались корректно, в `index.html` (в корне проекта) должна быть подключена соответствующая таблица стилей:

```html
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined&display=swap">
```

Без этой строки иконки (корзина, поиск, крестик и т.д.) будут отображаться как обычный текст вместо значков.

## API

Данные о товарах берутся с публичного mock-API:

```
https://6a3c21e6e4a07f202e167825.mockapi.io/smiles
```

Формат одного товара:

```json
{
  "id": "1",
  "title": ":)",
  "description": "Весёлый смайлик, наделяет улыбкой",
  "price": 10
}
```

## Хранение корзины

Содержимое корзины хранится в `localStorage` под ключом `smiles-shop:cart` в виде массива товаров с дополнительным полем `amount` (количество). Это позволяет корзине сохраняться между сессиями без необходимости в backend.
